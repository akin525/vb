API Setup

Airtime ,Internet & Utility API https://www.reloadly.com/registration

Create Account Create an account on https://www.reloadly.com/registration, verify your account and proceed to login.
Click on API key to reveal your test and live credentials

Setup Keys Open your source code and core folder, search for a file named .env, open to modify the file and update line 56 to 59 with your CLIENTID and SECRETKEY

Cable TV & Utility API https://www.vtpass.com/register

Create Account Create an account on https://www.vtpass.com/register, verify your account and proceed to login.

Setup Keys Open your source code and core folder, search for a file named .env, open to modify the file and update line 61 to 62 with your VTPASSUSERNAME and VTPASSPASSWORD

Please note, by default the API is set to TEST MODE on the source code please modify MODE=TEST from the .env file and change TEST to MODE=LIVE to switch to live mode.

Key Description Type
CLIENTIDLIVE Your live client id from reloadly LIVE
CLIENTSECRETLIVE Your live secret key from reloadly LIVE
CLIENTIDTEST Your test client id from reloadly TEST
CLIENTSECRETTEST Your test secret key from reloadly TEST
VTPASSUSERNAME Your login email to vtpass.com
VTPASSPASSWORD Your login password to vtpass.com
